AP Chemistry

Learn chemistry at three levels at once: macroscopic observations, particulate-level models, and symbolic equations/calculations. The path deliberately repeats stoichiometry, equilibrium thinking, energy, and particle interactions across units.

54lessons

Start here

Status: Hybrid digital AP Exam in 2026; course framework organized into 9 units.

Suggested pacing: 36 weeks; 9 units; 2 lessons/week with calculation drills.

Official course/reference page

Prerequisites

  • Algebra fluency
  • Unit conversions and scientific notation
  • Comfort with equations, graphs, and proportional reasoning

Course skills

  • Use models to explain chemical behavior at atomic, particulate, and macroscopic levels.
  • Calculate amounts, rates, equilibrium quantities, pH, thermodynamic values, and electrochemical quantities.
  • Interpret experimental procedures, graphs, spectra, titration data, and particulate diagrams.
  • Explain cause-and-effect using structure, bonding, energy, entropy, and particle interactions.
  • Write balanced equations, net ionic equations, and chemical explanations with units/significant figures.

1. Which molecule has a bent geometry despite having 4 electron groups?
A) CH₄
B) NH₃
C) H₂O
D) CO₂
C: H₂O.
H₂O has 4 electron groups (2 bonding + 2 lone pairs) → tetrahedral electron geometry but BENT molecular geometry. The lone pairs repel more strongly, compressing the bond angle to ~104.5°.
Units 4–7
Kinetics & Equilibrium
~25–30% of exam
⚗️
Kinetics — Rate Laws & Integrated Rate Laws
Rate Law:
rate = k[A]ᵐ[B]ⁿ
// orders from EXPERIMENT, never coefficients
Integrated Rate Laws:
Zero order:   [A] = [A]₀ − kt
// linear: [A] vs t
First order:  ln[A] = ln[A]₀ − kt
// linear: ln[A] vs t
Second order: 1/[A] = 1/[A]₀ + kt
// linear: 1/[A] vs t
Half-life:
First order:  t½ = 0.693/k
// CONSTANT — independent of [A]₀
Second order: t½ = 1/(k[A]₀)
// depends on initial concentration
Arrhenius:
k = Ae^(−Ea/RT)
// catalyst lowers Ea, increases rate
⚗️
Equilibrium & Le Chatelier's Principle
Le Chatelier's Principle
Add reactant / remove product:
shifts RIGHT (toward products)
Add product / remove reactant:
shifts LEFT
Increase pressure:
shifts toward fewer moles of gas
Increase temperature:
shifts toward endothermic side (treats heat as reactant/product)
Add catalyst:
equilibrium UNCHANGED — just reaches it faster. K does not change.
Add inert gas at constant volume:
NO shift — partial pressures unchanged
📝 Unit Quiz
0/0 correct
Reset
1. For N₂ + 3H₂ ⇌ 2NH₃ (ΔH=−92 kJ), which increases NH₃ yield?
A) Increasing temperature
B) Increasing pressure
C) Adding a catalyst
D) Decreasing pressure
B: Increasing pressure.
4 moles gas → 2 moles. Increasing pressure favors fewer gas moles → shifts right → more NH₃. Choice A would shift LEFT (exothermic reaction, adding heat shifts away from products). Catalyst (C) doesn't change yield.
2. Reaction orders are determined by:
A) Balanced equation stoichiometric coefficients
B) Experiment only
C) The rate-determining step only
D) Activation energy
B: Experiment only.
This is one of the most common exam misconceptions. Stoichiometric coefficients in the balanced equation NEVER directly give rate law orders (except in elementary reactions where the mechanism IS the stoichiometry).
Unit 8
Acids, Bases & Buffers
~11–15% of exam
⚗️
pH, Ka, and Henderson-Hasselbalch
pH = −log[H⁺]
// pOH = −log[OH⁻]  // pH + pOH = 14 at 25°C
Ka × Kb = Kw = 1×10⁻¹⁴
// at 25°C
Stronger acid = smaller Ka = smaller pKa
Weak acid ICE table:
HA ⇌ H⁺ + A⁻
I:  C    0    0
C: −x  +x   +x
E: C−x   x    x
Ka = x²/(C−x) ≈ x²/C
// 5% approximation valid when x << C
[H⁺] = x = √(Ka·C)
Henderson-Hasselbalch (buffer):
pH = pKa + log([A⁻]/[HA])
// Buffer most effective when pH ≈ pKa
📝 Unit Quiz
0/0 correct
Reset
1. At the half-equivalence point in a weak acid titration:
A) pH = 7
B) pH = pKa
C) pH = pKb
D) pH = 14
B: pH = pKa.
At half-equivalence, half the acid has been converted to its conjugate base: [HA] = [A⁻]. Substituting into H-H: pH = pKa + log(1) = pKa + 0 = pKa. This is used to FIND Ka experimentally from a titration curve.
Perfect Score Checklist
Reaction orders from EXPERIMENT — never stoichiometric coefficients
First-order half-life is CONSTANT: t½ = 0.693/k
Catalyst: lowers Ea, increases rate, does NOT change K or ΔG°
Le Chatelier: adding inert gas at constant volume = NO shift
pH = pKa at half-equivalence point
Henderson-Hasselbalch: pH = pKa + log([A⁻]/[HA])
Stronger acid = smaller Ka = more dissociation
H bonding requires H bonded to N, O, or F
VSEPR: lone pairs repel more than bonding pairs
Anode = oxidation (always); cathode = reduction (always)

<h3 class="sub"><span class="sub-icon">⚗️</span>Kinetics — Rate Laws & Integrated Rate Laws</h3><div class="code-block"><span class="hl">Rate Law:</span>   rate = k[A]ᵐ[B]ⁿ    <span class="cmt">// orders from EXPERIMENT, never coefficients</span>

<span class="hl">Integrated Rate Laws:</span>
Zero order:   [A] = [A]₀ − kt          <span class="cmt">// linear: [A] vs t</span>
First order:  ln[A] = ln[A]₀ − kt      <span class="cmt">// linear: ln[A] vs t</span>
Second order: 1/[A] = 1/[A]₀ + kt     <span class="cmt">// linear: 1/[A] vs t</span>

<span class="hl">Half-life:</span>
First order:  t½ = 0.693/k             <span class="cmt">// CONSTANT — independent of [A]₀</span>
Second order: t½ = 1/(k[A]₀)          <span class="cmt">// depends on initial concentration</span>

<span class="hl">Arrhenius:</span>  k = Ae^(−Ea/RT)         <span class="cmt">// catalyst lowers Ea, increases rate</span>
</div><h3 class="sub"><span class="sub-icon">⚗️</span>Equilibrium & Le Chatelier's Principle</h3><div class="callout"><div class="callout-lbl">Le Chatelier's Principle</div><ul><li><strong>Add reactant / remove product:</strong> shifts RIGHT (toward products)</li><li><strong>Add product / remove reactant:</strong> shifts LEFT</li><li><strong>Increase pressure:</strong> shifts toward fewer moles of gas</li><li><strong>Increase temperature:</strong> shifts toward endothermic side (treats heat as reactant/product)</li><li><strong>Add catalyst:</strong> equilibrium UNCHANGED — just reaches it faster. K does not change.</li><li><strong>Add inert gas at constant volume:</strong> NO shift — partial pressures unchanged</li></ul></div><div class="quiz-section">

<h3 class="sub"><span class="sub-icon">⚗️</span>pH, Ka, and Henderson-Hasselbalch</h3><div class="code-block"><span class="hl">pH = −log[H⁺]</span>            <span class="cmt">// pOH = −log[OH⁻]  // pH + pOH = 14 at 25°C</span>
<span class="hl">Ka × Kb = Kw = 1×10⁻¹⁴</span>    <span class="cmt">// at 25°C</span>
Stronger acid = smaller Ka = smaller pKa

<span class="hl">Weak acid ICE table:</span>
HA ⇌ H⁺ + A⁻
I:  C    0    0
C: −x  +x   +x
E: C−x   x    x
Ka = x²/(C−x) ≈ x²/C  <span class="cmt">// 5% approximation valid when x &lt;&lt; C</span>
[H⁺] = x = √(Ka·C)

<span class="hl">Henderson-Hasselbalch (buffer):</span>
pH = pKa + log([A⁻]/[HA])   <span class="cmt">// Buffer most effective when pH ≈ pKa</span>

<script>function pick(el, isCorrect, qId) {
  const qItem = el.closest('.q-item');
  if (qItem.classList.contains('answered')) return;
  qItem.classList.add('answered');
  el.classList.add(isCorrect ? 'correct' : 'wrong');
  if (!isCorrect) {
    qItem.querySelectorAll('.q-choice[data-correct="true"]').forEach(c => c.classList.add('show-correct'));
  }
  qItem.querySelectorAll('.q-choice').forEach(c => c.classList.add('picked'));
  const explain = document.getElementById('exp-' + qId);
  if (explain) explain.classList.add('show');
  // update score
  const quiz = el.closest('.quiz-section');
  const answered = quiz.querySelectorAll('.q-item.answered').length;
  const correct = quiz.querySelectorAll('.q-choice.correct').length;
  const total = quiz.querySelectorAll('.q-item').length;
  const scoreEl = quiz.querySelector('.quiz-score');
  if (scoreEl) scoreEl.textContent = correct + '/' + answered + ' correct (' + total + ' total)';
}
function resetQuiz(btn) {
  const quiz = btn.closest('.quiz-section');
  quiz.querySelectorAll('.q-item').forEach(q => {
    q.classList.remove('answered');
    q.querySelectorAll('.q-choice').forEach(c => c.classList.remove('correct','wrong','show-correct','picked'));
    q.querySelectorAll('.q-explain').forEach(e => e.classList.remove('show'));
  });
  const scoreEl = quiz.querySelector('.quiz-score');
  if (scoreEl) scoreEl.textContent = '0/0 correct';
}
