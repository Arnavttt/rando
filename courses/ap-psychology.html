AP Psychology

This path moves from research foundations to biology, cognition, development, social behavior, and mental health. Each lesson connects a term to an applied scenario and research evidence.

20lessons

Status: Fully digital AP Exam in 2026; revised framework has 5 units and science practices.

Suggested pacing: 18–36 weeks; 5 units; 2–3 lessons/week if semester course.

Official course/reference page

Prerequisites

  • Basic biology vocabulary helps
  • Ability to apply terms to scenarios
  • Attention to research method details

Course skills

  • Use psychological theories and concepts to explain behavior and mental processes.
  • Analyze research methods, variables, operational definitions, ethical issues, and data.
  • Differentiate correlation, experimentation, causation, sampling, and bias.
  • Apply biological, cognitive, social, developmental, and clinical explanations to scenarios.
  • Interpret results and limitations of psychological studies.

Research Methods
Correlation vs. Causation
Correlation
shows a relationship (r from −1 to +1). Does NOT prove causation.
Only experiments can establish causation.
Independent variable (IV): the variable manipulated.
Dependent variable (DV): the variable measured.
Control group: no manipulation. Experimental group: receives manipulation.
Double-blind: neither participants nor researchers know group assignment.

1. Which research method is the ONLY one that can establish cause-and-effect?
A) Case study
B) Survey
C) Experiment
D) Naturalistic observation
C: Experiment.
Only experiments manipulate an independent variable and control other factors, allowing causal conclusions. Correlational methods identify relationships but cannot rule out third variables or directionality.
2. Negative reinforcement is best defined as:
A) Punishment that decreases behavior
B) Removing an aversive stimulus to increase behavior
C) Adding a pleasant stimulus to increase behavior
D) Ignoring a behavior to decrease it
B is correct.
CRITICAL DISTINCTION: Negative reinforcement INCREASES behavior by REMOVING something unpleasant. It is NOT punishment. Example: taking Tylenol removes a headache (aversive stimulus), so you're more likely to take Tylenol again.

Brain Regions
Limbic System
Amygdala = fear/aggression/emotion. Hippocampus = NEW memory formation (not storage). Hypothalamus = hunger, thirst, body temperature, fight-or-flight trigger.
Cerebral Cortex Lobes
Frontal = planning, judgment, personality, motor strip. Parietal = touch, spatial. Occipital = vision. Temporal = hearing, language (Wernicke's area).
Brainstem
Medulla = breathing, heartbeat (survival). Pons = sleep, arousal. Cerebellum = balance, coordination, procedural memory.
Language Areas
Broca's area (frontal lobe) = SPEECH PRODUCTION. Damage → Broca's aphasia (halting speech). Wernicke's area (temporal lobe) = language COMPREHENSION. Damage → Wernicke's aphasia (fluent but meaningless).
📝 Unit Quiz
0/0 correct
Reset
1. The hippocampus is most associated with:
A) Fear and aggression
B) Formation of new long-term memories
C) Language production
D) Motor coordination
B is correct.
The hippocampus converts short-term memories into long-term ones. Patient H.M. had his hippocampus removed and developed anterograde amnesia — he could not form new explicit memories. His old memories were intact (stored elsewhere in the cortex).
2. The 'fight-or-flight' response is controlled by the:
A) Parasympathetic nervous system
B) Somatic nervous system
C) Sympathetic nervous system
D) Central nervous system
C: Sympathetic nervous system.
Sympathetic = fight-or-flight: increases heart rate, dilates pupils, inhibits digestion, releases adrenaline. Parasympathetic = rest-and-digest: reverses all these. An easy mnemonic: Sympathetic = 'Stress response.'

Classical Conditioning (Pavlov)
Classical Conditioning Terms
UCS
(Unconditioned Stimulus): naturally causes response (food → salivation).
UCR
(Unconditioned Response): natural response to UCS (salivation).
CS
(Conditioned Stimulus): neutral stimulus paired with UCS (bell).
CR
(Conditioned Response): learned response to CS alone (salivation to bell).
Extinction:
CR weakens when CS presented without UCS repeatedly.
Spontaneous Recovery:
Extinguished CR reappears after rest period.
Generalization:
Responding to stimuli similar to the CS.
Discrimination:
Responding only to specific CS, not similar stimuli.

Reinforcement Schedules
Fixed Ratio (FR):
After set number of responses. High, steady rate. (Every 10 items = bonus.)
Variable Ratio (VR):
Unpredictable number of responses. HIGHEST rate, MOST resistant to extinction. (Slot machines.)
Fixed Interval (FI):
After set time. Scallop pattern — slow then burst. (Weekly quiz.)
Interval (VI):
Unpredictable time. Steady, slow rate. (Checking email randomly.)

0/0 correct
Reset
1. A student cleans their room to stop their parent's nagging. This is an example of:
A) Positive reinforcement
B) Negative reinforcement
C) Positive punishment
D) Negative punishment
B: Negative reinforcement.
An aversive stimulus (nagging) is REMOVED when the behavior occurs (cleaning), making the behavior more likely in the future. Negative = remove. Reinforcement = increase behavior.
2. Which reinforcement schedule produces behavior most resistant to extinction?
A) Fixed ratio
B) Fixed interval
C) Variable ratio
D) Variable interval
C: Variable ratio.
Because reinforcement is unpredictable, the organism keeps responding hoping the next response will be reinforced — just like gambling. This is why slot machines are so addictive.

Fundamental Attribution Error
Overestimating personality/disposition and underestimating situational factors when explaining OTHERS' behavior. We explain our own with situation.
Cognitive Dissonance
Discomfort when beliefs and behavior conflict. We change beliefs to match behavior (post-purchase rationalization).
Social Facilitation
Presence of others improves performance on simple/well-learned tasks but hurts performance on difficult/novel tasks.
Groupthink
Group prioritizes harmony over critical thinking, leading to poor decisions. Symptoms: illusion of invulnerability, self-censorship.
📝 Unit Quiz
0/0 correct
Reset
1. The Fundamental Attribution Error states that we tend to:
A) Blame ourselves for others' failures
B) Overestimate situational factors when explaining others' behavior
C) Overestimate dispositional factors and underestimate situational factors when explaining OTHERS' behavior
D) Attribute our own failures to personality flaws
C is correct.
When we see someone trip, we think 'clumsy person' (disposition) rather than 'uneven sidewalk' (situation). For our own behavior, we do the opposite — we blame the situation. This is the actor-observer bias.

1. Schizophrenia is most associated with excess activity of which neurotransmitter?
A) Serotonin
B) GABA
C) Dopamine
D) Acetylcholine
C: Dopamine.
The dopamine hypothesis of schizophrenia holds that excess dopamine activity (especially in mesolimbic pathways) causes positive symptoms. Antipsychotic medications work by blocking dopamine D2 receptors.
2. Cognitive-Behavioral Therapy (CBT) primarily works by:
A) Prescribing medication to correct neurotransmitter imbalances
B) Identifying and challenging distorted thought patterns that lead to maladaptive behaviors
C) Exploring unconscious conflicts from childhood
D) Using classical conditioning to replace negative responses
B is correct.
CBT is based on the cognitive model: distorted thoughts (cognitive distortions like catastrophizing and all-or-nothing thinking) drive emotional distress and maladaptive behavior. By identifying and reframing these thoughts, symptoms improve.

<h3 class="sub"><span class="sub-icon">🔄</span>Classical Conditioning (Pavlov)</h3><div class="callout"><div class="callout-lbl">Classical Conditioning Terms</div><ul><li><strong>UCS</strong> (Unconditioned Stimulus): naturally causes response (food → salivation).</li><li><strong>UCR</strong> (Unconditioned Response): natural response to UCS (salivation).</li><li><strong>CS</strong> (Conditioned Stimulus): neutral stimulus paired with UCS (bell).</li><li><strong>CR</strong> (Conditioned Response): learned response to CS alone (salivation to bell).</li><li><strong>Extinction:</strong> CR weakens when CS presented without UCS repeatedly.</li><li><strong>Spontaneous Recovery:</strong> Extinguished CR reappears after rest period.</li><li><strong>Generalization:</strong> Responding to stimuli similar to the CS.</li><li><strong>Discrimination:</strong> Responding only to specific CS, not similar stimuli.</li></ul></div><h3 class="sub"><span class="sub-icon">🔄</span>Operant Conditioning (Skinner)</h3><table class="ctable"><tr><th>Type</th><th>What Happens</th><th>Effect on Behavior</th><th>Example</th></tr><tr><td>Positive Reinforcement</td><td>Add pleasant stimulus</td><td>INCREASES behavior</td><td>Gold star for good work</td></tr><tr><td>Negative Reinforcement</td><td>REMOVE unpleasant stimulus</td><td>INCREASES behavior</td><td>Taking painkiller removes headache → take more</td></tr><tr><td>Positive Punishment</td><td>Add unpleasant stimulus</td><td>DECREASES behavior</td><td>Speeding ticket</td></tr><tr><td>Negative Punishment</td><td>REMOVE pleasant stimulus</td><td>DECREASES behavior</td><td>Take away phone for bad behavior</td></tr></table><div class="callout"><div class="callout-lbl">Reinforcement Schedules</div><ul><li><strong>Fixed Ratio (FR):</strong> After set number of responses. High, steady rate. (Every 10 items = bonus.)</li><li><strong>Variable Ratio (VR):</strong> Unpredictable number of responses. HIGHEST rate, MOST resistant to extinction. (Slot machines.)</li><li><strong>Fixed Interval (FI):</strong> After set time. Scallop pattern — slow then burst. (Weekly quiz.)</li><li><strong>Interval (VI):</strong> Unpredictable time. Steady, slow rate. (Checking email randomly.)</li></ul></div><div class="quiz-section">

<li class="q-choice" data-correct="false" onclick="pick(this,false,'psy_l_q0')">D) Negative punishment</li>

<div class="q-explain show" id="exp-psy_l_q0" style="display:none"><strong>B: Negative reinforcement.</strong> An aversive stimulus (nagging) is REMOVED when the behavior occurs (cleaning), making the behavior more likely in the future. Negative = remove. Reinforcement = increase behavior.</div>
</div><div class="q-item">
<div class="q-text">2. Which reinforcement schedule produces behavior most resistant to extinction?</div>
<ul class="q-choices"><li class="q-choice" data-correct="false" onclick="pick(this,false,'psy_l_q1')">A) Fixed ratio</li>
<li class="q-choice" data-correct="false" onclick="pick(this,false,'psy_l_q1')">B) Fixed interval</li>
<li class="q-choice" data-correct="true" onclick="pick(this,true,'psy_l_q1')">C) Variable ratio</li>
<li class="q-choice" data-correct="false" onclick="pick(this,false,'psy_l_q1')">D) Variable interval</li>
</ul>
<div class="q-explain show" id="exp-psy_l_q1" style="display:none"><strong>C: Variable ratio.</strong> Because reinforcement is unpredictable, the organism keeps responding hoping the next response will be reinforced — just like gambling. This is why slot machines are so addictive.</div>
</div></div>

<h3 class="sub"><span class="sub-icon">👥</span>Classic Studies — All Must Know</h3><table class="ctable"><tr><th>Study</th><th>Researcher</th><th>Key Finding</th><th>Concept</th></tr><tr><td>Electric shock obedience</td><td>Milgram (1963)</td><td>65% gave max shock when authority figure present</td><td>Obedience to authority overrides personal morality</td></tr><tr><td>Stanford Prison Study</td><td>Zimbardo (1971)</td><td>Normal students rapidly adopted guard/prisoner roles</td><td>Situational power; deindividuation; role absorption</td></tr><tr><td>Line conformity</td><td>Asch (1951)</td><td>75% conformed to wrong group answer at least once</td><td>Normative social influence; social pressure</td></tr><tr><td>Bystander effect</td><td>Darley & Latané (1968)</td><td>More bystanders → less likely anyone helps</td><td>Diffusion of responsibility; pluralistic ignorance</td></tr><tr><td>Bobo Doll</td><td>Bandura (1961)</td><td>Children imitated aggressive adult model</td><td>Observational learning / social learning theory</td></tr></table><div class="card-row"><div class="card ac-top"><h4>Fundamental Attribution Error</h4><p>Overestimating personality/disposition and underestimating situational factors when explaining OTHERS' behavior. We explain our own with situation.</p></div><div class="card ac-top"><h4>Cognitive Dissonance</h4><p>Discomfort when beliefs and behavior conflict. We change beliefs to match behavior (post-purchase rationalization).</p></div><div class="card ac-top"><h4>Social Facilitation</h4><p>Presence of others improves performance on simple/well-learned tasks but hurts performance on difficult/novel tasks.</p></div><div class="card ac-top"><h4>Groupthink</h4><p>Group prioritizes harmony over critical thinking, leading to poor decisions. Symptoms: illusion of invulnerability, self-censorship.</p></div></div><div class="quiz-section">

<div class="quiz-body"><div class="q-item">
<div class="q-text">1. The Fundamental Attribution Error states that we tend to:</div>
<ul class="q-choices"><li class="q-choice" data-correct="false" onclick="pick(this,false,'psy_s_q0')">A) Blame ourselves for others' failures</li>
<li class="q-choice" data-correct="false" onclick="pick(this,false,'psy_s_q0')">B) Overestimate situational factors when explaining others' behavior</li>
<li class="q-choice" data-correct="true" onclick="pick(this,true,'psy_s_q0')">C) Overestimate dispositional factors and underestimate situational factors when explaining OTHERS' behavior</li>
<li class="q-choice" data-correct="false" onclick="pick(this,false,'psy_s_q0')">D) Attribute our own failures to personality flaws</li>
</ul>
<div class="q-explain show" id="exp-psy_s_q0" style="display:none"><strong>C is correct.</strong> When we see someone trip, we think 'clumsy person' (disposition) rather than 'uneven sidewalk' (situation). For our own behavior, we do the opposite — we blame the situation. This is the actor-observer bias.</div>

<div class="q-explain show" id="exp-psy_a_q1" style="display:none"><strong>B is correct.</strong> CBT is based on the cognitive model: distorted thoughts (cognitive distortions like catastrophizing and all-or-nothing thinking) drive emotional distress and maladaptive behavior. <script>function pick(el, isCorrect, qId) {
  const qItem = el.closest('.q-item');
  if (qItem.classList.contains('answered')) return;
  qItem.classList.add('answered');
  el.classList.add(isCorrect ? 'correct' : 'wrong');
  if (!isCorrect) {
    qItem.querySelectorAll('.q-choice[data-correct="true"]').forEach(c => c.classList.add('show-correct'));
  }
  qItem.querySelectorAll('.q-choice').forEach(c => c.classList.add('picked'));
  const explain = document.getElementById('exp-' + qId);
  if (explain) explain.classList.add('show');
  // update score
  const quiz = el.closest('.quiz-section');
  const answered = quiz.querySelectorAll('.q-item.answered').length;
  const correct = quiz.querySelectorAll('.q-choice.correct').length;
  const total = quiz.querySelectorAll('.q-item').length;
  const scoreEl = quiz.querySelector('.quiz-score');
  if (scoreEl) scoreEl.textContent = correct + '/' + answered + ' correct (' + total + ' total)';
}
function resetQuiz(btn) {
  const quiz = btn.closest('.quiz-section');
  quiz.querySelectorAll('.q-item').forEach(q => {
    q.classList.remove('answered');
    q.querySelectorAll('.q-choice').forEach(c => c.classList.remove('correct','wrong','show-correct','picked'));
    q.querySelectorAll('.q-explain').forEach(e => e.classList.remove('show'));
  });
  const scoreEl = quiz.querySelector('.quiz-score');
  if (scoreEl) scoreEl.textContent = '0/0 correct';
}
