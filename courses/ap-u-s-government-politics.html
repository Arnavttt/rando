AP U.S. Government & Politics

This path moves from constitutional foundations to institutions, rights, behavior, and participation. Each lesson connects principles to documents, cases, data, and argument writing.

Suggested pacing: 18–36 weeks; 5 units; 2 lessons/week if semester course.

Official course/reference page

Prerequisites

  • Basic U.S. civics vocabulary
  • Ability to read primary-source excerpts
  • Willingness to write evidence-based claims

Course skills

  • Describe and explain political principles, institutions, behavior, and processes.
  • Apply course concepts to political scenarios.
  • Analyze quantitative data, maps, charts, and political cartoons.
  • Analyze required foundational documents and Supreme Court decisions.
  • Write claim-evidence-reasoning arguments using required documents/cases.

1. Which case established judicial review?
A) McCulloch v. Maryland
B) Baker v. Carr
C) Marbury v. Madison
D) Gideon v. Wainwright
C: Marbury v. Madison (1803).
Chief Justice Marshall established that the Court has the power — and duty — to strike down laws that violate the Constitution.
2. Citizens United v. FEC (2010) held that:
A) Campaign contributions are unlimited
B) Corporations can directly donate to campaigns
C) Corporations and unions have First Amendment rights to make independent political expenditures
D) The government can regulate all political advertising
C is correct.
Independent expenditures (not coordinated with a campaign) by corporations and unions are protected First Amendment speech. The Court held political speech doesn't lose protection based on the speaker's corporate identity.

1. Federalist No. 51 primarily argues that tyranny is best prevented by:
A) A virtuous citizenry
B) Separation of powers and checks and balances — ambition counteracts ambition
C) Direct democracy
D) Judicial review
B is correct.
Madison argued constitutional structure — not virtue — is the key safeguard. Because men are not angels, government must be designed so institutions restrain each other.

<script>function pick(el, isCorrect, qId) {
  const qItem = el.closest('.q-item');
  if (qItem.classList.contains('answered')) return;
  qItem.classList.add('answered');
  el.classList.add(isCorrect ? 'correct' : 'wrong');
  if (!isCorrect) {
    qItem.querySelectorAll('.q-choice[data-correct="true"]').forEach(c => c.classList.add('show-correct'));
  }
  qItem.querySelectorAll('.q-choice').forEach(c => c.classList.add('picked'));
  const explain = document.getElementById('exp-' + qId);
  if (explain) explain.classList.add('show');
  // update score
  const quiz = el.closest('.quiz-section');
  const answered = quiz.querySelectorAll('.q-item.answered').length;
  const correct = quiz.querySelectorAll('.q-choice.correct').length;
  const total = quiz.querySelectorAll('.q-item').length;
  const scoreEl = quiz.querySelector('.quiz-score');
  if (scoreEl) scoreEl.textContent = correct + '/' + answered + ' correct (' + total + ' total)';
}
function resetQuiz(btn) {
  const quiz = btn.closest('.quiz-section');
  quiz.querySelectorAll('.q-item').forEach(q => {
    q.classList.remove('answered');
    q.querySelectorAll('.q-choice').forEach(c => c.classList.remove('correct','wrong','show-correct','picked'));
    q.querySelectorAll('.q-explain').forEach(e => e.classList.remove('show'));
  });
  const scoreEl = quiz.querySelector('.quiz-score');
  if (scoreEl) scoreEl.textContent = '0/0 correct';
}
