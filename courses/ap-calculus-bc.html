AP Calculus BC

This path builds calculus as a chain: limits support derivatives, derivatives support applications and integrals, integrals support accumulation, and BC adds sequences, series, parametric/polar/vector reasoning.

60lessons

Prerequisites

  • Strong algebra/trigonometry/precalculus
  • Function notation and graph behavior
  • Comfort with symbolic manipulation

Course skills

  • Understand limits, derivatives, integrals, differential equations, applications, parametric/polar/vector functions, and series.
  • Move between graphical, numerical, analytical, and verbal representations.
  • Justify conclusions using theorem hypotheses and mathematical reasoning.
  • Use calculator appropriately for graphing, solving, numeric derivative/integral.

Limits & Continuity
4–7% of exam

Limit Rules & Evaluating Limits
Indeterminate Forms & L'Hôpital's Rule
If a limit gives 0/0 or ∞/∞, use L'Hôpital's Rule: lim f/g = lim f'/g' (differentiate top and bottom separately).
Other indeterminate forms: 0·∞, ∞−∞, 0⁰, 1^∞, ∞⁰ — rewrite algebraically before applying L'Hôpital.
Continuity:
f is continuous at c if: (1) f(c) exists, (2) lim f(x) exists, (3) lim f(x) = f(c).
Intermediate Value Theorem: if f is continuous on [a,b] and N is between f(a) and f(b), then f(c)=N for some c in (a,b).

1. If f(x) = x³·eˣ, then f'(x) =
A) 3x²·eˣ
B) x³·eˣ

Integration

Integration Techniques
Technique
When to Use
Formula/Method
Basic Antiderivatives
Simple polynomials, trig, exp
∫xⁿ dx = xⁿ⁺¹/(n+1)+C; ∫eˣ dx = eˣ+C; ∫sinx dx = −cosx+C
u-Substitution
Composition of functions inside integral
Let u = inner function; du = derivative × dx. Replace and integrate.
Integration by Parts
Product of two functions
∫u dv = uv − ∫v du. LIATE for choosing u: Logarithm, Inverse trig, Algebraic, Trig, Exponential.
Trig Substitution
Integrals with √(a²−x²), √(a²+x²), √(x²−a²)
Substitute x=a sinθ, a tanθ, or a secθ respectively.
Partial Fractions
Rational functions with factorable denominator
Decompose into simpler fractions; integrate each.
Fundamental Theorem of Calculus
FTC Part 1:
If F(x) = ∫[a to x] f(t) dt, then F'(x) = f(x)
FTC Part 2:
∫[a to b] f(x) dx = F(b) − F(a) where F is any antiderivative of f
Average value
of f on [a,b]: f_avg = (1/(b−a)) ∫[a to b] f(x) dx

1. ∫ 2x·e^(x²) dx =
A) 2e^(x²) + C

BC: Series
Sequences, Series & Taylor
17–18% BC
∫
Convergence Tests — All 7
Test
Use When
Result
Divergence Test
Always try first
If lim(aₙ)≠0, diverges. If = 0, inconclusive.
Integral Test
aₙ=f(n), f continuous/decreasing/positive
Σaₙ and ∫f(x)dx have same convergence behavior
p-Series
Σ 1/nᵖ
Converges if p > 1; diverges if p ≤ 1
Geometric Series
Σ arⁿ
Converges if |r|<1, sum=a/(1−r); diverges if |r|≥1
Direct Comparison
Similar to known series
If 0≤aₙ≤bₙ: bₙ converges → aₙ converges; aₙ diverges → bₙ diverges
Limit Comparison
Ratio of terms has finite nonzero limit
If lim(aₙ/bₙ)=L>0, both series same behavior
Ratio Test
Factorials, exponentials
If lim|aₙ₊₁/aₙ|<1: converges; >1: diverges; =1: inconclusive
Alternating Series Test
Alternating signs
Converges if: terms decrease AND approach 0
Taylor & Maclaurin Series — Must Memorize
eˣ = 1 + x + x²/2! + x³/3! + ... = Σ xⁿ/n!
(all x)
sin x = x − x³/3! + x⁵/5! − ... = Σ (−1)ⁿx^(2n+1)/(2n+1)!
(all x)
cos x = 1 − x²/2! + x⁴/4! − ... = Σ (−1)ⁿx^(2n)/(2n)!
(all x)
1/(1−x) = 1 + x + x² + x³ + ... = Σ xⁿ
(|x|<1)
Taylor series centered at a: f(x) = Σ f⁽ⁿ⁾(a)/n! · (x−a)ⁿ

1. The series Σ 1/n² converges because:
A) It's a geometric series
B) It's a p-series with p=2>1
C) The ratio test gives L<1

If = 0, inconclusive.</td></tr><tr><td>Integral Test</td><td>aₙ=f(n), f continuous/decreasing/positive</td><td>Σaₙ and ∫f(x)dx have same convergence behavior</td></tr><tr><td>p-Series</td><td>Σ 1/nᵖ</td><td>Converges if p > 1; diverges if p ≤ 1</td></tr><tr><td>Geometric Series</td><td>Σ arⁿ</td><td>Converges if |r|<1, sum=a/(1−r); diverges if |r|≥1</td></tr><tr><td>Direct Comparison</td><td>Similar to known series</td><td>If 0≤aₙ≤bₙ: bₙ converges → aₙ converges; aₙ diverges → bₙ diverges</td></tr><tr><td>Limit Comparison</td><td>Ratio of terms has finite nonzero limit</td><td>If lim(aₙ/bₙ)=L>0, both series same behavior</td></tr><tr><td>Ratio Test</td><td>Factorials, exponentials</td><td>If lim|aₙ₊₁/aₙ|<1: converges; >1: diverges; =1: inconclusive</td></tr><tr><td>Alternating Series Test</td><td>Alternating signs</td><td>Converges if: terms decrease AND approach 0</td></tr></table><div class="callout"><div class="callout-lbl">Taylor & Maclaurin Series — Must Memorize</div><ul><li><strong>eˣ = 1 + x + x²/2! + x³/3! + ... = Σ xⁿ/n!</strong>  (all x)</li><li><strong>sin x = x − x³/3! + x⁵/5! − ... = Σ (−1)ⁿx^(2n+1)/(2n+1)!</strong>  (all x)</li><li><strong>cos x = 1 − x²/2! + x⁴/4! − ... = Σ (−1)ⁿx^(2n)/(2n)!</strong>  (all x)</li><li><strong>1/(1−x) = 1 + x + x² + x³ + ... = Σ xⁿ</strong>  (|x|<1)</li><li>Taylor series centered at a: f(x) = Σ f⁽ⁿ⁾(a)/n! · (x−a)ⁿ</li></ul></div><div class="quiz-section">
