📐Order of Operations (PEMDAS)

PEMDAS / BODMAS
  • Parentheses / Brackets first
  • Exponents (powers and roots)
  • Multiplication and Division left to right (equal priority)
  • Addition and Subtraction left to right (equal priority)
  • Example: 3 + 4 × 2² − (1 + 2) = 3 + 4×4 − 3 = 3 + 16 − 3 = 16

Unit 2
Functions
Core Concept

📐What is a Function?

Function Definition
  • A function assigns exactly one output to each input.
  • Notation: f(x) — read 'f of x'. x is the input (independent variable). f(x) is the output (dependent variable).
  • Vertical Line Test: A graph represents a function if and only if every vertical line crosses it at most once.
  • Domain: All valid inputs (x-values). Range: All resulting outputs (y-values).

1

3

📝 Unit Quiz 0/0 correct
1. The domain of f(x) = √(x + 4) is:
  • A) x > −4
  • B) x ≥ −4
  • C) x ≥ 4
  • D) All real numbers
B: x ≥ −4. Even roots require non-negative radicand: x + 4 ≥ 0 → x ≥ −4. Domain includes −4 (square root of 0 = 0, which is valid). Interval notation: [−4, ∞).
2. If f(x) = x², then g(x) = (x−3)² + 2 represents:
  • A) Shift right 3, up 2
  • B) Shift left 3, up 2
  • C) Shift right 3, down 2
  • D) Stretch by factor 3, up 2
A: Shift right 3, up 2. f(x−h) shifts RIGHT by h (the minus inside moves right). Adding 2 outside shifts UP by 2. The vertex of f(x)=x² is at (0,0); the vertex of g(x) is at (3,2). NOTE: Many students confuse the sign — f(x−3) goes RIGHT, not left.
Unit 3
Polynomial & Rational Functions
Key Unit

📐Polynomials — Key Properties

Degree, Leading Term & End Behavior
  • Degree: highest exponent. Leading coefficient: coefficient of highest-degree term.
  • End behavior (what happens as x→±∞):
  • Even degree, POSITIVE leading coeff: both ends go UP (↑ ↑)
  • Even degree, NEGATIVE leading coeff: both ends go DOWN (↓ ↓)
  • Odd degree, POSITIVE leading coeff: left DOWN, right UP (↓ ↑)
  • Odd degree, NEGATIVE leading coeff: left UP, right DOWN (↑ ↓)

📐Finding Zeros of Polynomials

Zero-Finding Strategy
1
Factor the polynomial if possible: GCF first, then factor quadratic/other patterns.
2
Rational Zero Theorem: Possible rational zeros = ±(factors of constant term)/(factors of leading coefficient)
3
Synthetic division: Test possible rational zeros. If remainder = 0, it's a zero.
4
Multiplicity matters: If (x−a)^n is a factor: n=odd → graph CROSSES x-axis at a; n=even → graph TOUCHES (bounces) at a.
5
Complex zeros: Non-real complex zeros come in conjugate pairs.

📐Rational Functions — Asymptotes

Asymptote TypeHow to FindExample
Vertical Asymptote (VA)Set denominator = 0 (after canceling common factors)f(x)=1/(x−3): VA at x=3
Horizontal Asymptote (HA)Compare degrees of num & denom. If deg(top) < deg(bot): HA y=0. If equal: HA y = leading coefficients ratio. If deg(top) > deg(bot): NO HA.f(x)=(2x²)/(3x²+1): HA y=2/3
Slant Asymptotedeg(num) = deg(denom)+1: divide polynomialsf(x)=(x²+1)/(x): y=x
HoleFactor cancels from both num & denom: hole at that x-valuef(x)=(x²−1)/(x−1): hole at x=1
📝 Unit Quiz 0/0 correct
1. The polynomial f(x) = −2x⁵ + 3x² − 1 has end behavior:
  • A) Both ends up
  • B) Left up, right down
  • C) Left down, right up
  • D) Both ends down
B: Left up, right down. Odd degree (5) + negative leading coefficient (−2): as x→−∞, f(x)→+∞ (left end UP); as x→+∞, f(x)→−∞ (right end DOWN). Rule: odd degree = ends go opposite directions. Negative leading = right end points DOWN.
2. For f(x) = (x²−4)/(x²−x−2), identify any holes, vertical asymptotes, and horizontal asymptotes.
  • A) VA: x=2, HA: y=1, no holes
  • B) Hole at x=2, VA: x=−1, HA: y=1
  • C) VA: x=2 and x=−1, HA: y=1
  • D) Hole at x=−1, VA: x=2, HA: y=0
B is correct. Factor: numerator = (x−2)(x+2); denominator = (x−2)(x+1). The factor (x−2) cancels → HOLE at x=2. Remaining: (x+2)/(x+1). VA: set x+1=0 → x=−1. HA: degrees equal, ratio of leading coefficients = 1/1 = 1 → y=1.
Unit 4
Exponential & Logarithmic Functions
Key Unit

📐Exponential Functions

Exponential Function: f(x) = a·bˣ
  • b > 1: Growth function — increases as x increases
  • 0 < b < 1: Decay function — decreases as x increases
  • Domain: All real numbers. Range: y > 0 (always positive)
  • Horizontal asymptote: y = 0
  • The number e ≈ 2.718 — base of natural exponential: f(x) = eˣ

📐Logarithms — Definition & Properties

log_b(x) = y means bʸ = x
  • log₁₀(x) = log(x) — common logarithm
  • logₑ(x) = ln(x) — natural logarithm
  • Product rule: log_b(xy) = log_b(x) + log_b(y)
  • Quotient rule: log_b(x/y) = log_b(x) − log_b(y)
  • Power rule: log_b(xⁿ) = n·log_b(x)
  • Change of base: log_b(x) = log(x)/log(b) = ln(x)/ln(b)
Step-by-Step: Solve 3^(2x+1) = 27
1
Recognize 27 = 3³. Rewrite: 3^(2x+1) = 3³
2
Same base → exponents equal: 2x + 1 = 3
3
Solve: 2x = 2, x = 1
4
Check: 3^(2·1+1) = 3³ = 27 ✓
Step-by-Step: Solve log₂(x+3) + log₂(x−1) = 5
1
Combine using product rule: log₂[(x+3)(x−1)] = 5
2
Convert to exponential: (x+3)(x−1) = 2⁵ = 32
3
Expand: x² + 2x − 3 = 32
4
Rearrange: x² + 2x − 35 = 0
5
Factor: (x+7)(x−5) = 0 → x = −7 or x = 5
6
Check domain: x+3 > 0 and x−1 > 0 require x > 1. So x = −7 is EXTRANEOUS. Answer: x = 5
📝 Unit Quiz 0/0 correct
1. log₃(81) = ?
  • A) 27
  • B) 4
  • C) 3
  • D) 9
B: 4. log₃(81) = y means 3ʸ = 81. Since 3⁴ = 81, y = 4. Always restate as 'base to what power equals the argument?'
2. Solve: 2^(x−1) = 16
  • A) x = 4
  • B) x = 5
  • C) x = 3
  • D) x = 8
B: x = 5. 16 = 2⁴. So 2^(x−1) = 2⁴ → x−1 = 4 → x = 5. Check: 2^(5−1) = 2⁴ = 16 ✓

📐Solving Systems — Three Methods

Method 1: Substitution
1
Solve one equation for one variable (choose the easiest).
2
Substitute that expression into the other equation.
3
Solve for the remaining variable.
4
Back-substitute to find the first variable.
5
Check in BOTH original equations.
Method 2: Elimination
1
Arrange both equations in standard form (Ax + By = C).
2
Multiply one or both equations to make coefficients of one variable opposites.
3
Add equations to eliminate that variable.
4
Solve for remaining variable, back-substitute.

📝 Unit Quiz 0/0 correct
1. Solve the system: 2x + y = 7, x − y = 2
  • A) x=3, y=1
  • B) x=2, y=3
  • C) x=3, y=0
  • D) x=4, y=2
A: x=3, y=1. Elimination: Add both equations: 3x = 9 → x = 3. Substitute into x−y=2: 3−y=2 → y=1. Check in 2x+y=7: 2(3)+1 = 7 ✓
2. For the 2×2 matrix [[3,2],[1,4]], the determinant is:
  • A) 14
  • B) 10
  • C) 4
  • D) 6
B: 10. det([[a,b],[c,d]]) = ad − bc = (3)(4) − (2)(1) = 12 − 2 = 10.

📝 Unit Quiz 0/0 correct
1. An arithmetic sequence has a₁ = 3 and d = 5. What is a₇?
  • A) 33
  • B) 35
  • C) 38
  • D) 30
A: 33. aₙ = a₁ + (n−1)d. a₇ = 3 + (7−1)·5 = 3 + 30 = 33. Arithmetic sequences add the common difference each term.
2. The equation x² + y² − 6x + 4y + 9 = 0 represents a circle with center:
  • A) (3, −2)
  • B) (−3, 2)
  • C) (6, −4)
  • D) (3, 2)
A: (3, −2). Complete the square: (x²−6x+9) + (y²+4y+4) = −9+9+4 → (x−3)² + (y+2)² = 4. Center (h,k) = (3,−2), r = 2.
Perfect Score Checklist
  • Even root → radicand ≥ 0. Denominator → ≠ 0. Logarithm → argument > 0.
  • f(x−h) shifts RIGHT by h (not left — the minus inside is counterintuitive)
  • Vertical asymptote: set denominator = 0. Check for holes (cancelable factors) first.
  • Horizontal asymptote: degrees equal → ratio of leading coefficients; top degree < bottom → y=0; top > bottom → no HA.
  • log_b(x) = y means bʸ = x — always restate as an exponential.
  • Product/quotient/power rules for logs — expand or condense logs on every exam.
  • Check extraneous solutions in logarithmic equations — plug back in to verify domain.
  • Determinant of 2×2: ad − bc
  • Arithmetic: add/subtract d each term. Geometric: multiply by r each term.
  • Complete the square to find circle/ellipse/parabola center from general form.
