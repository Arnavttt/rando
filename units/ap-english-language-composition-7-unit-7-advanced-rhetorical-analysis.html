Advanced Rhetorical Analysis — Lesson 1: analyzing purpose at multiple levels; rhetorical shifts across a passage

Unit 7 · Lesson 1
Learn on this site

The core question is never “what happened?” but “how do choices create meaning, argument, tone, structure, or effect for a reader/audience?”

analyzing purpose at multiple levels

analyzing purpose at multiple levels is a reading-and-writing skill. Learn what to notice in a text, how to name the author’s or writer’s choice, and how to explain the effect of that choice on meaning, argument, audience, tone, or structure.

Test yourself: teach this idea from memory, give one example, identify one trap, and answer one exam-style question about it.
