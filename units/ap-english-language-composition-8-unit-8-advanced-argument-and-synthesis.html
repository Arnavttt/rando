Advanced Argument and Synthesis — Lesson 1: multi-paragraph line of reasoning; strategic concession and refutation

Unit 8 · Lesson 1
Learn on this site

The core question is never “what happened?” but “how do choices create meaning, argument, tone, structure, or effect for a reader/audience?”

multi-paragraph line of reasoning

multi-paragraph line of reasoning is a reading-and-writing skill. Learn what to notice in a text, how to name the author’s or writer’s choice, and how to explain the effect of that choice on meaning, argument, audience, tone, or structure.

Test yourself: teach this idea from memory, give one example, identify one trap, and answer one exam-style question about it.

strategic concession and refutation

strategic concession and refutation is a reading-and-writing skill. Learn what to notice in a text, how to name the author’s or writer’s choice, and how to explain the effect of that choice on meaning, argument, audience, tone, or structure.

Test yourself: teach this idea from memory, give one example, identify one trap, and answer one exam-style question about it.

connecting examples across domains

connecting examples across domains is a reading-and-writing skill. Learn what to notice in a text, how to name the author’s or writer’s choice, and how to explain the effect of that choice on meaning, argument, audience, tone, or structure.

Test yourself: teach this idea from memory, give one example, identify one trap, and answer one exam-style question about it.

Objectives

How this can be tested
  • analyze choices and effects
  • write a defensible thesis

  • Revise one sentence for clarity and sophistication.
