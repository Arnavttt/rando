Argument Development — Lesson 1: building an original argument; counterargument and concession/refutation

Unit 3 · Lesson 1
Learn on this site

The core question is never “what happened?” but “how do choices create meaning, argument, tone, structure, or effect for a reader/audience?”

building an original argument

building an original argument is a reading-and-writing skill. Learn what to notice in a text, how to name the author’s or writer’s choice, and how to explain the effect of that choice on meaning, argument, audience, tone, or structure.

Test yourself: teach this idea from memory, give one example, identify one trap, and answer one exam-style question about it.

counterargument and concession/refutation

counterargument and concession/refutation is a reading-and-writing skill. Learn what to notice in a text, how to name the author’s or writer’s choice, and how to explain the effect of that choice on meaning, argument, audience, tone, or structure.

Test yourself: teach this idea from memory, give one example, identify one trap, and answer one exam-style question about it.

logical fallacies and weak reasoning

logical fallacies and weak reasoning is a reading-and-writing skill. Learn what to notice in a text, how to name the author’s or writer’s choice, and how to explain the effect of that choice on meaning, argument, audience, tone, or structure.

Test yourself: teach this idea from memory, give one example, identify one trap, and answer one exam-style question about it.

Objectives

How this can be tested
  • analyze choices and effects
  • write a defensible thesis

  • explain commentary

Practice before moving on

  • Choose two pieces of evidence and explain how each supports the claim.
  • Revise one sentence for clarity and sophistication.
